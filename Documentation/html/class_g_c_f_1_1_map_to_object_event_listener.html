Detailed Description

When set as event listener on GCF::MapToObject the corresponding virtual functions in this class will be called when a object pointer is added, removed or deleted from the GCF::MapToObject.

To set this as event listener of GCF::MapToObject you need to use the method GCF::MapToObject::setEventListener(GCF::MapToObjectEventListener* listener).
