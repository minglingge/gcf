GCF 3 Documentation

Generic Component Framework (GCF) - is a Qt based C++ library that empowers developers and organizations to leverage a well-designed component framework for building large applications (on Windows, Linux and Mac) using Qt. With GCF your applications enjoy enhanced maintainability, testability and extensibility.
