Modules
Here is a list of all modules:
 GCF Core ModuleThe "GCF Core Module" is the foundation of all modules in GCF. The module works on both Qt 4.8 and on Qt 5
 GCF's GDriveLite component
GCF's GDriveLite is no longer guaranteed to work in 2019. The source code & documentation is provided only if you want to use it as an inspiraton to build your own stack. We strongly recommend that you use Qt Network Authorization to enable Google Drive API access in your applications. By default GCF's GDriveLite component and examples are not compiled along with GCF. You have to explicitly enable compilation of these modules.
 GCF Gui ModuleThe "GCF Gui Module" extends GCF Core Module to provide support for QtGui+QtWidgets based applications. The module works on both Qt 4.8 and on Qt 5
 IPC module in GCFThe "GCF IPC Module" extends offers support for inter-process communication in your GCF applications
 GCF Quick ModuleThe "GCF Quick Module" extends GCF Core Module to provide support for QtGui+QtQuick based applications. The module works only on Qt 5
